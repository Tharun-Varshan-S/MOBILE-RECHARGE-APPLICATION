# Mobile Recharge Application

A modern, responsive React frontend for a mobile recharge application, inspired by Airtel Thanks and Paytm.

## Features

- **User Portal**:
  - Recharge Plans browsing and filtering
  - Secure Payment Mockup (UPI, Card, Wallet)
  - Dashboard with data usage and validity tracking
  - Transaction History
- **Admin Portal**:
  - Dedicated Login
  - Dashboard with key metrics
  - Plan Management (CRUD operations)
  - System-wide Notifications
- **Voice Assistant**: Touch-to-speak UI simulation.

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- React Router v7

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Project Structure

- `src/components`: Reusable UI components
- `src/pages`: Application pages
- `src/layout`: Main layout wrapper
- `src/context`: Authentication context
- `src/data`: Mock data
